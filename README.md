### `npm start`

Gõ npm start ở terminal để chạy dự án trong chế độ phát triển. Trình duyệt sẽ mở trang mặc định [http://localhost:3000].
Tự động reload khi có sự thay đổi.

### `npm run build`

Tạo ra folder Build chứa tất cả code đã được mã hoá, và tăng trải nghiệm người dùng thực tế.

Các file cùng ngôn ngữ sẽ được gom thành một file. Tối ưu code và lược bớt khoảng cách không cần thiết khi phát triển.
Biết thêm thông tin ở đây [deployment](https://facebook.github.io/create-react-app/docs/deployment)
